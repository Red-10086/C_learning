# 45 天计算机前置学习计划（进 Java 班前）
## 第一周：C 语言收尾与内存基础
| 天数 | 核心任务 | 具体执行 |
|------|----------|----------|
| 第 1 天 | 文件操作收尾 | 完成《明解 C 语言》入门篇第 13 章剩余练习（至少 2 题），必须用到 `fseek` 和二进制读写 |
| 第 2 天 | 动态内存初探 | 阅读《明解 C 语言》中级篇第 5 章（`malloc`/`free`），手写第一个动态分配示例 |
| 第 3 天 | 栈与堆的直觉建立 | 阅读《程序是怎样跑起来的》第 4-5 章（内存的结构），画出栈和堆的内存分区图 |
| 第 4 天 | 计组内存布局预习 | 阅读《计算机是怎样跑起来的》第 3 章，重点看内存部分 |
| 第 5 天 | 本周总结 | 整理文件操作安全不等式、`malloc`/`free` 边界、栈与堆的本质区别 |

---

## 第二周：内存布局深化与 OS 内存管理入门
| 天数 | 核心任务 | 具体执行 |
|------|----------|----------|
| 第 6 天 | 结构体与内存对齐 | 亲手测试不同成员顺序的结构体大小（用 `sizeof`），理解填充字节的存在 |
| 第 7 天 | 指针与内存地址的物理直觉 | 用 `%p` 打印多个变量的地址，观察栈上地址分配规律；用 `malloc` 打印堆地址，对比差异 |
| 第 8 天 | OS 内存管理极简入门 | 阅读《操作系统导论》前 2 章，一句话搞懂“虚拟内存” |
| 第 9 天 | 指针与动态数据结构的关系 | 手写一个最简单的链表节点，用 `malloc` 创建 2 个节点并链接 |
| 第 10 天 | 本周总结 | 整理内存对齐、虚拟内存、链表与堆的关系等公理 |

---

## 第三周：算法与数据结构印象建立
| 天数 | 核心任务 | 具体执行 |
|------|----------|----------|
| 第 11 天 | 算法书入门 | 阅读《我的第一本算法书》第 1-2 章：链表、数组、栈、队列，只看图解 |
| 第 12 天 | 继续阅读 | 第 3-4 章：哈希表、二分查找，只看图解 |
| 第 13 天 | 继续阅读 | 第 5-7 章：排序、树、堆，快速浏览 |
| 第 14 天 | 印象总结 | 为每个数据结构画一张草图，写一句核心操作 |
| 第 15 天 | 本周总结 | 整理算法与数据结构公理 |

---

## 第四周：用 C 语言验证算法印象
| 天数 | 核心任务 | 具体执行 |
|------|----------|----------|
| 第 16 天 | 手写链表 | 用 `struct` 和 `malloc` 实现单向链表，含创建、遍历、释放 |
| 第 17 天 | 手写栈 | 用数组或链表实现栈，含 push、pop、peek |
| 第 18 天 | 手写二分查找 | 用递归和非递归两种方式实现，测试边界情况 |
| 第 19 天 | 手写哈希表 | 用数组+链表实现最简哈希表，理解碰撞处理 |
| 第 20 天 | 本周总结 | 整理手动实现与图解印象之间的认知修正 |

---

## 第五周：网络协议栈基础
| 天数 | 核心任务 | 具体执行 |
|------|----------|----------|
| 第 21 天 | 网络分层模型 | 阅读《网络是怎样连接的》第 1 章前半部分 |
| 第 22 天 | IP 地址与端口 | 搞懂 IP 地址和端口的区别，用 `ping` 验证 |
| 第 23 天 | TCP 三次握手 | 画出时序图，一句话解释“为什么是三次” |
| 第 24 天 | HTTP 协议入门 | 用 Chrome 开发者工具看真实请求/响应 |
| 第 25 天 | 本周总结 | 整理网络分层、TCP 握手、HTTP 请求/响应公理 |

---

## 第六周：MySQL 基础与正则表达式速通
| 天数 | 核心任务 | 具体执行 |
|------|----------|----------|
| 第 26 天 | MySQL 环境搭建 | 安装 MySQL，用命令行登录，创建第一个数据库和表 |
| 第 27 天 | CRUD 操作 | 阅读《MySQL 必知必会》第 3-5 章，亲手执行增删改查 |
| 第 28 天 | 过滤与排序 | 阅读第 6-8 章，学习 `WHERE`、`ORDER BY`、`LIKE` |
| 第 29 天 | 正则表达式速通 | 阅读《正则表达式必知必会》前 3 章，在在线平台上测试 |
| 第 30 天 | 本周总结 | 整理 MySQL 和正则表达式公理 |

---

## 第七周：Java 环境预习与网页基础
| 天数 | 核心任务 | 具体执行 |
|------|----------|----------|
| 第 31 天 | 安装 Java 环境 | 下载 JDK 17，用命令行运行 `HelloWorld.java` |
| 第 32 天 | Java 与 C 编译模型对比 | 画出对比图，一句话描述 JVM |
| 第 33 天 | HTML 极简入门 | 手写一个包含标题、段落、表格的静态网页 |
| 第 34 天 | HTTP 与表单交互 | 在 HTML 中添加表单，理解请求参数传递 |
| 第 35 天 | 本周总结 | 整理 Java 环境、编译模型、HTML 基础 |

## 第八周：弹性缓冲、总复习与进班准备
| 天数 | 核心任务 | 具体执行 |
|------|----------|----------|
| 第 36-39 天 | 弹性缓冲 | 回顾前七周薄弱或感兴趣部分，查漏补缺 |
| 第 40-42 天 | 总复习与知识库整理 | 翻阅 60 天公理化笔记，为每个模块写“面试可讲内容” |
| 第 43-45 天 | 最终温习 | 不再学新内容，默写关键公理，形成稳定知识索引 |
