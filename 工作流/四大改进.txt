## 二、四大支柱详解

### 🏃 支柱 1：锻炼 - 节节高模式

| 原则 | 具体内容 |
| :--- | :--- |
| **核心节奏** | **练一休一**（不追高、不连续） |
| **运动量** | 4-5km 快走（累了就慢走） |
| **恢复标准** | 运动后 24 小时内恢复，否则减量 |
| **加量条件** | 当前强度完全无压力持续 2 周以上 |
| **停止信号** | 肢端冰凉、乏力>24 小时、脑雾散不掉 |
| **心态** | "芝麻开花节节高"，以月计/半年计 |

**关键认知：** 锻炼是"借贷"，休息是"还款"。**只借不还=猝死风险**。

---

### 👁️ 支柱 2：护眼 - 翻页 + 纸质工作流

| 原则 | 具体内容 |
| :--- | :--- |
| **屏幕上限** | **≤2-3 小时/天**（医生硬性要求） |
| **显示模式** | 禁用滚轮，改用**翻页模式**（Page Down/Page Up） |
| **学习载体** | 纸质教材为主（《明解 C 语言》），屏幕只用于验证 |
| **休息频率** | 每 25 分钟远眺 20 秒（20-20-20 法则） |
| **睡前保护** | 睡前 1 小时不看屏幕 |
| **监控信号** | 眼睛干涩/多泪/复视→当天停止用屏幕 |

**关键认知：** 滚轮=低强度版 CS2，**翻页=真正的阅读**。

---

### ✋ 支柱 3：护手 - 每日手指假

| 原则 | 具体内容 |
| :--- | :--- |
| **核心问题** | 手从童年开始几乎**从未真正休息过** |
| **每日手指假** | **2-3 小时/天**手完全休息（分时段） |
| **快走时** | **不带手机**，手自然摆动 |
| **学习时** | 每 25 分钟手离开键盘/书 5 分钟（手放腿上） |
| **睡前** | 热水泡手 5-10 分钟 |
| **替代方案** | 语音输入代替打字，听书代替看书 |

**关键认知：** 手休息时的"欣快感"证明**手是核心疲劳源**。

---

### 🧠 支柱 4：注意力 - 听代替刷

| 原则 | 具体内容 |
| :--- | :--- |
| **核心问题** | 大脑难以降频，彻底放空太痛苦 |
| **过渡方案** | **听硬核狠人/有声书**（无为之心，只听故事） |
| **刷视频** | 不禁止，但**锁屏只听**，减少视觉刺激 |
| **时间控制** | 单次≤40 分钟，每天≤3 集 |
| **睡前 1 小时** | 不听刺激性内容，可听轻音乐/白噪音 |
| **心态** | "下台阶"，不追求一口气降到底 |

**关键认知：** 不是"戒掉刺激"，是**"换低能耗刺激"**。

├── 晚上：泡手后不碰东西 约 2 小时

├── 听硬核狠人：≤3 集（锁屏）
├── 刷视频：≤30 分钟（锁屏）

## 四、停止信号系统（四大支柱共用）

| 支柱 | 警戒信号 | 行动 |
| :--- | :--- | :--- |
| **锻炼** | 乏力>24 小时、肢端冰凉 | 暂停 1-3 天，减量 50% |
| **护眼** | 眼睛干涩/多泪/复视 | 当天停止用屏幕 |
| **护手** | 手指酸痛、精细控制困难 | 当天手指假延长至 4 小时 |
| **注意力** | 脑雾散不掉、失控感 | 停止学习，只听音频/发呆 |

**通用原则：**
```
出现 1 个信号 → 减量 50%
出现 2 个信号 → 暂停 1-2 天
出现 3 个信号 → 彻底休息 3-5 天
```
